1. XAMPP veya uygun bir PHP sunucusu kurunuz.
2. Apache ve MySQL servislerini başlatınız.
3. phpMyAdmin üzerinden "sadesurucukursu_db.sql" dosyasını içe aktarınız.
4. conn.php dosyasındaki;

   $servername
   $username
   $password
   $dbname

   bilgilerini kendi sunucunuza göre düzenleyiniz.
5. Projeyi çalıştırınız.
